# Carlos Manuel Torre Sada Portfolio

Static portfolio website built for GitHub Pages. It uses plain HTML, CSS, and JavaScript, so there is no build step or package installation.

## Local Preview

Open `index.html` directly in a browser. The project pages are linked from the landing page.

## GitHub Pages Deployment

1. Push this folder to a GitHub repository.
2. In the repository, open `Settings` > `Pages`.
3. Set the source to the default branch and the root folder.
4. Save the Pages settings and wait for GitHub to publish the site.

All links are relative so the site works from a GitHub Pages project URL.
